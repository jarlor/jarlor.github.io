LaTeX Environment Build

Latex Projects can be build quicker and better in Linux System. Based on this situation, build a Latex Env in WSL is the best plan. We can choose TeX Live as Latex Env, but it is too big to deploy it quickly and easy. TinyTeX is a better choose because of smaller size and easier to deploy. It’s important to note that we need to install an R environment(R Env) and master how to quickly install missing packages.
